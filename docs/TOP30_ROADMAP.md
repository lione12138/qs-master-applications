# QS Top 30 data roadmap

The target is not merely 30 admissions links. Each university progresses
through four independently measured stages:

1. official graduate admissions entry located
2. application-window granularity verified
3. representative master's programmes selected
4. exact, scoped application windows verified

## Batch 1 · QS 1–5

- MIT
- Imperial College London
- Stanford University
- University of Oxford
- Harvard University

Status: window-policy verification complete. The next task is selecting five
representative master's programmes or programme groups per institution.

Current official guidance does not support exact 2027 calendar records:

- MIT: applications generally become available in early fall; departments
  determine their own timelines.
- Stanford: the 2027 application is expected to open in mid-September 2026;
  deadlines are programme-specific.
- Oxford: guidance for applications made in 2026–27 is expected from
  mid-September 2026; course pages control the detailed deadlines.
- Harvard Griffin GSAS: applications become available in September and
  deadlines vary by programme.
- Imperial: the model is mixed, with shared departmental rounds, rolling
  availability, and course exceptions.

## Per-batch completion gate

A policy batch is complete when all five institutions have an official source,
verified granularity model, verification date, and any available next-cycle
guidance. Exact-window coverage is reported separately and is never inferred
from approximate month text.
